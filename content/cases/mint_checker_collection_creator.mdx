# Mint Checker Collection Creator: Solana NFT Telegram Bot

## The Challenge

- **No easy way to analyze and track Solana NFT collections and wallets in Telegram**
- **Manual checks are slow and error-prone**

## The Solution

A Python Telegram bot that:
- Validates Solana addresses and Tensor links
- Fetches and analyzes NFT collection data
- Summarizes and reports in Telegram chats
- Uses Selenium for web automation and aiogram for bot logic

## Technical Implementation

- **Python/aiogram**: Telegram bot logic
- **selenium**: Web automation for data extraction
- **requests**: API calls
- **solana-py**: Blockchain data

## Results

- **Automated NFT collection analysis**
- **Telegram chat integration**
- **Wallet and contract validation**

## Key Features

- Solana address and contract validation
- Tensor link support
- Telegram chat commands
- Automated progress updates
