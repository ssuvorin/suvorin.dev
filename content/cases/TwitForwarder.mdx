# TwitForwarder: Telegram Bot for X (Twitter) Monitoring

## The Challenge

- **No easy way to monitor X (Twitter) accounts and forward tweets to Telegram**
- **Manual checking is slow and unreliable**

## The Solution

A Python Telegram bot that:
- Tracks multiple X (Twitter) accounts
- Fetches latest tweets using Playwright and TLS client
- Sends new tweet notifications to Telegram chats
- Stores tracking info in SQLite

## Technical Implementation

- **Python/aiogram**: Telegram bot logic
- **playwright**: Headless browser for tweet fetching
- **bs4**: HTML parsing
- **sqlite3**: Local database for tracking

## Results

- **Automated tweet monitoring**
- **Telegram notifications for new tweets**
- **Multi-account support**

## Key Features

- Add/remove tracked accounts
- Real-time tweet forwarding
- Manual and automated update checks

## Technology Stack

- Python

- sqlite3
- bs4
