# Mindshare Analyzer: Kaito AI Project Growth

## The Challenge

- **No automated way to track mindshare growth of project leaders**
- **Manual analysis is slow and error-prone**

## The Solution

A Python tool that:
- Analyzes top-100 leaders for any project
- Tracks mindshare growth over 5+ weeks
- Generates detailed Excel and JSON reports
- Supports interactive and CLI usage

## Technical Implementation

- **Python**: Core logic and data processing
- **FastAPI**: Optional web interface
- **requests**: Data fetching
- **openpyxl**: Excel report generation
- **Jinja2**: Templating for reports

## Results

- **Automated mindshare analytics**
- **Excel and JSON reports**
- **Supports 50+ projects from tickers.json**

## Key Features

- Top-100 leader analysis
- Multi-week growth tracking
- Excel/JSON export
- Interactive CLI and web UI

## Technology Stack

- Python
- FastAPI
- requests
- openpyxl
- Jinja2
